## 新设备登录

如果你已经在一个设备上登录，并成功导出了密钥，那么当你用新设备登录的时候，软件就会自动向您索要这个密钥。

新设备登录可以是任何形式的新登录请求，当你用一个新的登录环境或者清除缓存数据的浏览器、客户端登录NerChat!的时候，这些环境会被视为一个新的登录设备，关于设备等内容可以参考
